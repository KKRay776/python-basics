# My Python Practice Scripts

This is a simple repository where I am saving all my basic Python programs as I learn. I wanted a central place to keep track of my progress, and also a way to easily share my code with friends who are learning Python too.

I tried to keep the code clean and commented so it's easy to follow along.

## What's in here?
It is mostly small scripts covering the essentials:
- Basic logic (if/else statements and loops)
- Working with lists and dictionaries
- Writing simple functions 
- Basic problem-solving exercises (like simple calculators or text games)

## How to use this
If you want to run any of these programs on your own computer:

1. Clone this repository:
   git clone https://github.com/KKRay776/python-basics.git

2. Open your terminal, go into the project folder, and run whichever file you want using Python:
   cd python-basics
   python filename.py
